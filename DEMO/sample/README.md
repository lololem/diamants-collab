# DIAMANTS Sample Demonstrations

This directory contains interactive demos for the DIAMANTS drone simulation system.

## Available Demos

| Demo | File | Description |
|------|------|-------------|
| **Swarm Intelligence** | `Frontend_Swarm_Intelligence_Demo/` | Complete interactive swarm behavior demo |

## Frontend Swarm Intelligence Demo

A standalone HTML demo showcasing the DIAMANTS swarm intelligence system:
- Flocking and swarming behaviors
- Multi-agent coordination
- Collective exploration patterns
- Real-time intelligence metrics

### How to Run

Open the HTML file directly in a browser or use a local HTTP server:

```bash
# Using VS Code Live Server or similar
# Open Frontend_Swarm_Intelligence_Demo/DIAMANTS_Complete_Frontend_Swarm_Intelligence_Demo.html
```
